Sentiment Analysis

Social Media NLP Pipeline

The Challenge

A retail brand needed to monitor real-time public sentiment on Twitter to detect potential PR crises early. Manual monitoring was impossible due to the volume of data, and existing tools lacked the domain-specific nuance required to distinguish dissatisfaction from sarcasm.

The Solution

I developed a custom NLP pipeline using TensorFlow and BERTopic to ingest, classify, and visualize social media sentiment in real-time:

  • Data Ingestion: Built a scalable scraper using Twitter API v2 and Kafka for high-throughput stream processing.
  • Custom Model: Fine-tuned a BERT model on domain-specific datasets to achieve high accuracy in detecting sarcasm and mixed sentiments.
  • Dashboard: Created an interactive Streamlit dashboard for stakeholders to view trends and alert triggers.

Technical Architecture

  • Model: HuggingFace Transformers (BERT), NLTK, Spacy
  • Backend: FastAPI for model serving, Redis for caching
  • Database: PostgreSQL for structured data, Elasticsearch for search
  • Visualization: Streamlit, Plotly

The Result

The system achieved 90% classification accuracy and reduced response time to negative PR incidents by 85%. It currently processes over 50,000 tweets daily for the client.
